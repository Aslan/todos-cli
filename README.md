[![chat on gitter](https://badges.gitter.im/pepebecker.svg)](https://gitter.im/pepebecker)

## Usage

### Print Todos

```shell
cargo run get
```

### Add Todo

```shell
cargo run add "Do something fun"
```

### Toggle Todo

```shell
cargo run toggle 0
```

### Remove Todo

```shell
cargo run remove 0
```
